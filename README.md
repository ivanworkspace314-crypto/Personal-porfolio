Personal portfolio website built with the Next.js App Router and Tailwind CSS.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- ESLint

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Project Structure

```text
app/                # App Router pages and layouts
components/         # Shared UI components
public/             # Static assets
```

## Deployment

Build and start the production server:

```bash
npm run build
npm run start
```

This project is compatible with Vercel and any Node hosting that supports Next.js.
